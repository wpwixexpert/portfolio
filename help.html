Program Help

To Preview HTML file, click button in main toolbar or use Ctrl+P hotkey on external keyboard. If you click preview button during editing JS or CSS file, Preview Window will open index.html from the same folder. Also, you can configure another Preview Root: open Recent Files, long-click on a HTML or PHP file, and select "Set As Preview File"

To open HTML, CSS or Javascript Quick Reference, long-click button in main toolbar.

To open Code Snippets, long-click [""] button in main toolbar.

To open/close Recent Files list, click filename row in main window or use Ctrl+R hotkey on external keyboard.

To show/hide Main Toolbar, click button.

To fill the first item from Code Completion list, press menu button on your device or press Enter on external keyboard.

To Find / Replace or quick Jump to Line, click on button on bottom-right corner, and click button.

To Clear Recent Files list, open Recent Files, long-click on a file, and select "Clear Recent List".

External Keyboard Hotkeys

Ctrl+Z - undo
Ctrl+Y - redo
Ctrl+A - select all
Ctrl+C - copy
Ctrl+V - paste
Ctrl+F - find
Ctrl+H - replace
Ctrl+G - go to line
