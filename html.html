HTML quick reference

A - Anchor

<A HREF=URI REL=LinkTypes TYPE=ContentType TARGET=[ _blank | _self | _parent | _top ] TABINDEX=Number ONFOCUS=Script ONBLUR=Script> ... </A>

B - Bold Text

<B> ... </B>

BODY - Document Body

<BODY BACKGROUND=URI BGCOLOR=#RRGGBB TEXT=#RRGGBB ONLOAD=Script ONUNLOAD=Script> ... </BODY>

BR - Line Break

<BR CLEAR=[ left | all | right | none ]>

BUTTON - Submit, Reset, or Push Button

<BUTTON NAME=CDATA VALUE=CDATA TYPE=[ submit | reset | button ] DISABLED ACCESSKEY=Character ONFOCUS=Script ONBLUR=Script> ... </BUTTON>

DIV - Generic Block-level Container

<DIV ALIGN=[ left | center | right | justify ]> ... </DIV>

FONT - Font Change

<FONT SIZE=CDATA COLOR=#RRGGBB FACE=CDATA> ... </FONT>

FORM - Interactive Form

<FORM ACTION=URI METHOD=[ get | post ] ENCTYPE=ContentType NAME=CDATA ACCEPT-CHARSET=Charsets ACCEPT=ContentTypes ONSUBMIT=Script ONRESET=Script> ... </FORM>


<FORM METHOD=post ACTION="/test.php">
  <FIELDSET>
    <INPUT TYPE=text id=name ID=name></INPUT>
    <INPUT TYPE=text id=email ID=email></INPUT>
    <TEXTAREA id=adr ID=addr ROWS=4 COLS=40></TEXTAREA>  
  </FIELDSET>
  <INPUT TYPE=submit VALUE="Submit"></INPUT>
</FORM>

HR - Horizontal Rule

<HR ALIGN=[ left | center | right ] NOSHADE SIZE=Pixels WIDTH=Length>

IMG - Inline Image

<IMG SRC=URI ALT=Text LONGDESC=URI WIDTH=Length HEIGHT=Length USEMAP=URI ISMAP ALIGN=[ top | middle | bottom | left | right ] BORDER=Pixels HSPACE=Pixels VSPACE=Pixels NAME=CDATA>

INPUT - Form Input

<INPUT TYPE=[ text | password | checkbox | radio | submit | reset | file | hidden | image | button ] NAME=CDATA VALUE=CDATA CHECKED SIZE=CDATA MAXLENGTH=Number SRC=URIforImage ALT=CDATA DISABLED READONLY ACCEPT=ContentTypes ACCESSKEY=Character TABINDEX=Number ONFOCUS=Script ONBLUR=Script ONSELECT=Script ONCHANGE=Script>

Example

LI - List Item for <UL> (unordered list) or <OL> (ordered list)

<LI TYPE=[ disc | square | circle | 1 | a | A | i | I ] VALUE=Number> ... </LI>

LINK - Document Relationship

<LINK REL=[ Prev | Next | Start |... ] REV=[ Prev | Next | Start |... ] HREF=URI TYPE=[ "text/css" | "application/pdf" | ... ] TARGET=[ _blank | _self | _parent | _top ] MEDIA= [ screen | print | all | ... ] HREFLANG=LanguageCode CHARSET=Charset>

META - Metadata

<META NAME=[ description | keywords | robots | author | ... ] HTTP-EQUIV=[ Expires | Refresh | ... ] CONTENT=[ noindex | follow | nofollow |... ] SCHEME=CDATA>

OL - Ordered List (container for <LI>)

<OL TYPE=TYPE=[ 1 | a | A | i | I ] START=Number COMPACT> ... </OL>

OPTION - Menu Option (for <SELECT>)

<OPTION VALUE=CDATA SELECTED DISABLED LABEL=Text> ... </OPTION>

Example

P - Paragraph

<P ALIGN=[ left | center | right | justify ]> ... </P>

SCRIPT - Embedded Script

<SCRIPT TYPE=[ "text/javascript" | ... ] SRC=URI CHARSET=Charset DEFER> ... </SCRIPT>

SELECT - Option Selector (container for <OPTION>)

<SELECT NAME=CDATA MULTIPLE SIZE=Number DISABLED TABINDEX=Number ONFOCUS=Script ONBLUR=Script ONCHANGE=Script> ... </SELECT>


<SELECT id=browser>
  <OPTION LABEL="Firefox">Firefox</OPTION>
  <OPTION LABEL="MSIE">MSIE</OPTION>
  <OPTION LABEL="Opera">Opera</OPTION>
</SELECT>

STYLE - Embedded Style Sheet

<STYLE TYPE=[ "text/css" | ... ] MEDIA= [ screen | print | all | ... ] TITLE=Text> ... </STYLE>

TABLE - Table

<TABLE SUMMARY=Text WIDTH=Length BORDER=Pixels FRAME=[ void | above | below | hsides | lhs | rhs | vsides | box | border ] RULES=[ none | groups | rows | cols | all ] CELLSPACING=Length CELLPADDING=Length> ... </TABLE>


<TABLE BORDER=1 CELLPADDING=5>
  <TR ALIGN=left>
    <TD VALIGN=middle>One<TD>
    <TD VALIGN=middle>Two<TD>
    <TD VALIGN=middle>Three<TD>
  </TR>
</TABLE>

TD - Table Data Cell

<TD ROWSPAN=Number COLSPAN=Number SCOPE=[ row | col | rowgroup | colgroup ] AXIS=CDATA ALIGN=[ left | center | right | justify | char ] CHAR=AlignmentCharacter CHAROFF=Length VALIGN=[ top | middle | bottom | baseline ]> ... </TD>

Example

TR - Table Row

<TRALIGN=[ left | center | right | justify | char ] CHAR=AlignmentCharacter CHAROFF=Length VALIGN=[ top | middle | bottom | baseline ]> ... </TR>

Example

TEXTAREA - Multi-line Text Area for <FORM>

<TEXTAREA NAME=CDATA ROWS=Number COLS=Number DISABLED READONLY ACCESSKEY=Character TABINDEX=Number ONFOCUS=Script ONBLUR=Script ONSELECT=Script ONCHANGE=Script> ... </TEXTAREA>

Example

UL - Unordered List (container for <LI>)

<UL TYPE=TYPE=[ disc | square | circle ] COMPACT> ... </UL>
